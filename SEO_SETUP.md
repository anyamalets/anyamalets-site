# SEO Setup для anyamalets.ru

## ✅ Сделано в коде

- **sitemap.xml** — динамический из Sanity (`/app/sitemap.ts`)
- **robots.txt** — настроен с исключением `/studio/` и `/thanks`
- **Canonical URL** — `https://anyamalets.ru` в layout.tsx
- **Open Graph** — настроены для соцсетей
- **Редирект www** — 301 редирект с `www.anyamalets.ru` на `https://anyamalets.ru`

---

## 📋 Подробная инструкция: регистрация в поисковых системах

### 1. **Google Search Console** — пошагово

#### Шаг 1: Открыть GSC
Перейди на https://search.google.com/search-console (нужна учётная запись Google).

#### Шаг 2: Добавить собственность
- Нажми **+ Добавить свойство** (кнопка вверху слева)
- **Выбери тип собственности:**
  - ✅ **Domain property** (рекомендуется) — введи `anyamalets.ru` (без www, без https://)
  - Или **URL prefix** — введи `https://anyamalets.ru`

#### Шаг 3: Подтвердить собственность
После добавления сайта в Google Search Console автоматически откроется модальное окно **Подтвердить собственность**.

Вверху этого окна будут вкладки разных методов. Выбери **один способ**:

**Вариант A: DNS (рекомендуется)**
1. Нажми на вкладку **DNS** (если её нет, ищи в выпадающем меню)
2. Скопируй TXT запись (вид: `google-site-verification=xxxxx`)
3. Перейди в админ-панель **регистратора домена** (где куплен anyamalets.ru)
4. Найди раздел управления DNS (названия могут отличаться):

   | Регистратор | Где это находится |
   |---|---|
   | **Reg.ru** | Мои домены → anyamalets.ru → Управление → DNS → Добавить запись |
   | **Яндекс.Домены** | Домены → anyamalets.ru → Управление DNS |
   | **Namecheap** | Domain List → anyamalets.ru → Advanced DNS → Add New Record |
   | **GoDaddy** | Domains → anyamalets.ru → DNS → Add |
   | **1&1 (Ionos)** | Мои домены → anyamalets.ru → Управление DNS |
   | **Beget** | Домены → anyamalets.ru → DNS |

5. Добавь новую TXT запись:
   - **Type**: `TXT`
   - **Name** (или Host): `@` (или оставь пусто)
   - **Value** (или Data): вставь скопированную строку `google-site-verification=xxxxx`
   - **TTL**: `3600` (или по умолчанию)
6. Сохрани. Подожди 5-15 минут (пока DNS распространится)
7. В GSC нажми **Подтвердить** (вверху справа в окне подтверждения)

**📌 Если домен в Яндекс.Домены — как открыть админ-панель:**
1. Перейди на https://domains.yandex.ru/
2. Вход через учётную запись Яндекса
3. Найди `anyamalets.ru` в списке
4. Нажми на название домена
5. В левом меню выбери **Управление доменом** → **Управление DNS** (или просто **DNS**)
6. Добавляй TXT запись прямо отсюда (кнопка **+ Добавить запись**)

**Вариант B: HTML-метатег** (если DNS недоступен)
1. Скопируй метатег (вид: `<meta name="google-site-verification" content="xxxxx">`)
2. В файле [layout.tsx](app/layout.tsx) добавь в раздел `<head>`:
   ```tsx
   // Добавить в Metadata.verification
   verification: {
     google: "xxxxx" // вставить свой код
   }
   ```
3. Задеплой изменения на Vercel
4. В GSC нажми **Подтвердить**

#### Шаг 4: Добавить sitemap
После подтверждения собственности:
1. В левом меню найди **Sitemap**
2. Введи URL: `https://anyamalets.ru/sitemap.xml`
3. Нажми **Отправить**
4. Подожди 5-10 мин, обнови страницу
5. Должно появиться сообщение: "Получена" с числом страниц

#### Шаг 5: Проверить индексацию
1. Меню → **Отчёты** → **Охват**
2. Должно быть:
   - ✅ **Отправлено в индекс** — положительное число
   - ❌ **Исключено** — только `/studio/` и `/thanks` (это нормально)
   - ❌ **Ошибки** — должно быть 0

**Если индексация идёт медленно:**
- Нажми на статью в меню → **Запросить индексацию** (вверху справа)
- Это ускорит добавление в индекс Google

### 2. **Яндекс Вебмастер** — пошагово

#### Шаг 1: Открыть Вебмастер
Перейди на https://webmaster.yandex.ru/ (нужна учётная запись Яндекса).

#### Шаг 2: Добавить сайт
1. Нажми **Добавить сайт** (в левом меню)
2. Введи: `anyamalets.ru` (без www, без https://)
3. Нажми **Добавить**

#### Шаг 3: Подтвердить собственность
Яндекс сразу покажет окно **Подтвердить права на сайт** с вариантами методов.

Выбери **один способ**:

**Вариант A: DNS (рекомендуется)**
1. Скопируй TXT запись (вид: `yandex-verification: xxxxx`)
2. Перейди в админ-панель **регистратора домена** (где куплен anyamalets.ru):

   | Регистратор | Где это находится |
   |---|---|
   | **Reg.ru** | Мои домены → anyamalets.ru → Управление → DNS → Добавить запись |
   | **Яндекс.Домены** | Домены → anyamalets.ru → Управление DNS |
   | **Namecheap** | Domain List → anyamalets.ru → Advanced DNS → Add New Record |
   | **GoDaddy** | Domains → anyamalets.ru → DNS → Add |
   | **1&1 (Ionos)** | Мои домены → anyamalets.ru → Управление DNS |
   | **Beget** | Домены → anyamalets.ru → DNS |

3. Добавь новую TXT запись:
   - **Type**: `TXT`
   - **Name** (или Host): `@` (или оставь пусто)
   - **Value** (или Data): вставь скопированную строку
   - **TTL**: `3600` (или по умолчанию)
4. Сохрани. Подожди 5-15 минут (пока DNS распространится)
5. В Вебмастере нажми **Проверить**

**Вариант B: HTML-метатег**
1. Скопируй код вида `<meta name="yandex-verification" content="xxxxx">`
2. В [layout.tsx](app/layout.tsx) добавь в Metadata:
   ```tsx
   verification: {
     yandex: "xxxxx"
   }
   ```
3. Задеплой и нажми **Проверить**

#### Шаг 4: Добавить карту сайта
1. В левом меню выбери сайт → **Карта сайта**
2. Нажми **Добавить карту сайта**
3. Введи: `https://anyamalets.ru/sitemap.xml`
4. Нажми **Добавить**
5. Должно появиться значение в колонке **Получено** (кол-во страниц)

#### Шаг 5: Проверить индексацию
1. Меню → **Индексирование** (или **Статистика**)
2. Должны быть положительные числа в графе "Проиндексировано"
3. Обновляется 1-2 раза в день

### 3. **Vercel редирект www → non-www** (опционально)

Если хочешь убедиться, что редирект работает безупречно, в Vercel:

1. Перейди в проект → **Settings** → **Domains**
2. Нажми **Add** и добавь `www.anyamalets.ru`
3. После добавления нажми на домен → **Edit**
4. Поставь галку **Redirect to anyamalets.ru**
5. Выбери **Permanent (301)** в dropdown
6. Сохрани

Теперь все запросы `www.anyamalets.ru/*` перенаправляются на `anyamalets.ru/*` с кодом 301.

## ❓ FAQ

**В: Почему sitemap не показывает все статьи?**  
О: Динамический sitemap берёт только опубликованные статьи из Sanity. Если не видишь свою статью:
1. Проверь в Sanity Studio, опубликована ли статья (статус ≠ Draft)
2. Открой https://anyamalets.ru/sitemap.xml и перезагрузи (Ctrl+R)
3. Жди ~5 мин кеша Sanity

**В: Как добавить ещё страницы в sitemap?**  
О: Отредактируй `/app/sitemap.ts` и добавь новые URL в возвращаемый массив. Например:
```ts
{
  url: "https://anyamalets.ru/programs/rpp",
  lastModified: new Date().toISOString(),
  changeFrequency: "monthly",
  priority: 0.7,
}
```

**В: Сколько времени ждать, пока поисковики начнут ранжировать мой сайт?**  
О: Индексация — не то же самое, что ранжирование. Первые 1-3 месяца рейтинг растёт медленно. Ускоряется через:
- Регулярные качественные статьи (раз в неделю)
- Ссылки с авторитетных сайтов
- Социальные сигналы (репосты, лайки)

**В: Нужно ли добавлять sitemap вручную в robots.txt?**  
О: Нет, уже добавлено автоматически в `/app/robots.ts`:
```ts
sitemap: "https://anyamalets.ru/sitemap.xml"
```
